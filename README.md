## 5. 기술 스택
- Frontend: HTML/CSS/JS
- Backend: FastAPI
- Database: PostgreSQL, SQLAlchemy, Alembic
    - `sqlalchemy`: ORM (Object-Relational Mapping) 라이브러리로, Python 코드로 SQL 쿼리를 작성할 수 있습니다.
    - `alembic`: 데이터베이스 마이그레이션 도구로, 테이블 생성/변경 등의 작업을 쉽게 관리할 수 있습니다.
